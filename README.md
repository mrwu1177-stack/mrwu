# HelloYan - 智能加密货币策略分析系统

HelloYan 是一个基于 Next.js 的加密货币智能策略分析系统，提供实时市场监控、多币种策略分析和数据可视化功能。

## 功能特性

- 异动信号监控
- 多币种策略分析
- 布林带分析
- 币种排行榜
- 新闻追踪
- 市场概览
- 市场异动监控

## 技术栈

- Next.js 14
- React 18
- Tailwind CSS
- TradingView Lightweight Charts

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

## 部署

### Railway 部署

1. 连接 GitHub 仓库到 Railway
2. Railway 会自动检测 Next.js 项目
3. 配置环境变量（如需要）
4. 自动构建和部署

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0
